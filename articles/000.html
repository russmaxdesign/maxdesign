The required attribute in Chrome's accessibility tree

Which elements are defined as required in Chrome's accessibility tree?

All of the form controls below are defined using the required attribute. Each of these controls should be defined in Chrome's accessibility tree as: "Required: true"

The <input type="checkbox"> element:

  • DOES NOT register the required attribute in Chrome's accessibility tree
  • INCORRECTLY registers as "Invalid user entry: true" in Chrome's accessibility tree
  • DOES get announced as "Required" by VoiceOver
  • INCORRECTLY gets announced as "Invalid data" by VoiceOver
<input id="checkbox" type="checkbox" required>
<label for="checkbox">Subscribe</label>
