Services

We bring you extensive experience in planning, designing and developing websites, offering a complete service from concept to completion.

Whether your site is large or small, our team of specialists will make sure it is structured the right way and has the right content to attract visitors and keep them reading.

Training

Max Design works with clients to train staff in aspects of Responsive Web Design, CSS, HTML and Accessibility.

User Experience design

User experience design is the process of enhancing user satisfaction by improving the usability, ease of use, and pleasure provided in the interaction between the user and the product. User Experience design includes the visual design, the information architecture, structure and labelling, usability and more. We work with clients to develop websites and web applications that focusses on the users journey through the site or application.

User testing

Usability Testing is the process of observing people using websites and web applications - to determine if the product is intuitive and easy to use. We work with clients to test websites and web applications throughout all stages of the development cycle.

Responsive web design

Responsive Web Design is the process of designing and developing websites and web applications that adapt to any device - from wide screen desktop environments to tablets and mobile phones.

Front end development

Front-end Development is the process of building websites and web applications using HTML, CSS, JavaScript and related technologies. We focus on a pattern-library approach - defining components so that they can be reused across any website or web application. We use OOCSS and SMACSS principles. We use BEM-based naming conventions. We work in SASS and LESS.
