Courses

1. WCAG 2.0 Accessibility Compliance

Audience:
Web Testers, User Experience Professionals, Web Designers, Front-end Developers
Description:
A detailed overview of WCAG2 and how it affects websites today. Learn how to run compliance checks across your website and how to resolve key issues.

2. Accessible applications

Audience:
User Experience Professionals, Web Designers, Front-end Developers
Description:
Learn how to make complex application components accessible including in-page tabs, modals, dropdowns, error messages, dynamic content and more.

3. Introduction to WAI-ARIA

Audience:
User Experience Professionals, Web Designers, Front-end Developers
Description:
Learn all about WAI ARIA including understanding roles, states and properties, a detailed overview of important attributes, and how to make web components more accessible using ARIA.

4. Introduction to HTML

Audience:
Web Designers, Junior Front-end Developers
Description:
Learning about basic elements and attributes. Learn how to create accessible markup including building accessible tables and forms. Learn about valid code, semantic markup and web standards.

5. Introduction to HTML5

Audience:
Web Designers, Front-end Developers
Description:
Introduction to HTML5. Find out how it can be used in real websites today. Introduction to all the new HTML5 layout elements and HTML5 form elements and attributes. Introduction to HTML5 video and how to create native video across all modern browsers.

6. Introduction to RWD

Audience:
User Experience professionals, Web Designers, Front-end Developers
Description:
Responsive Web Design is now an essential skill for every web designer and developer. Learn RWD from the ground up so that you can create beautiful websites that look fantastic on any device. This course covers the basics of RWD including: Responsive type, Media queries, Creating simple fluid images, Grid systems and boilerplate, Building responsive layouts.

7. Real world RWD

Audience:
User Experience professionals, Web Designers, Front-end Developers
Description:
This course covers: Responsive navigation, Fluid video, Responsive forms, Responsive tables, Responsive images and Retrofit an existing site.

8. Introduction to CSS

Audience:
Web Designers, Junior Front-end Developers
Description:
Learning about linking CSS to HTML files, rulesets, selectors, basic styling, inheritance and cascade.

9. Hands-on CSS3

Audience:
Web Designers Front-end Developers
Description:
Learning what is CSS3, learning some of the key basics, CSS3 colors and selectors, text-shadow, box shadow, border-radius, gradients, font-face.
